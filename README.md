# Seba's Blog
Welcome to my open source blog


## 🛠 Usage
1. Install the dependencies with [pnpm](https://github.com/pnpm/pnpm):
  ```bash
  pnpm i
  ```
2. Run local dev server:
  ```bash
  pnpm dev
  ```
3. Build website:
  ```bash
  pnpm build
  ```
